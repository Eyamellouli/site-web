# Rapport — Projet Programmation Web 1 : Site web personnel

## 1. Objectif
Réaliser un site web personnel composé de plusieurs pages HTML reliées par un menu de navigation, un fichier CSS commun et du JavaScript (quiz).

## 2. Arborescence du projet
```
site_personnel_eya/
├─ index.html
├─ competences.html
├─ experiences.html
├─ parcours.html
├─ matieres.html
├─ passions.html
├─ manifestations.html
├─ quiz.html
├─ contact.html
├─ temoignage-iit.html
├─ assets/
│  ├─ css/
│  │  └─ style.css
│  ├─ js/
│  │  ├─ app.js
│  │  └─ quiz.js
│  └─ img/
└─ README.md
```

## 3. Description des pages
- **Accueil** : présentation, bio, compétences clés, raccourcis.
- **Parcours** : diplômes/formation.
- **Compétences** : langages, web, mobile, stack de stage.
- **Expériences** : stages + projets.
- **Manifestations** : événements / formations.
- **Matières** : TP/projets + zone à compléter.
- **Passions** : qualités + zone à compléter.
- **Quiz** : 12 questions, affichage note + réponses correctes.
- **Contact** : formulaire (mailto) + coordonnées.
- **Témoignage IIT** : texte court.
