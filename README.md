# Site personnel — Eya Mellouli (Programmation Web 1)

## Contenu
- **index.html** : accueil + bio
- **parcours.html** : parcours académique
- **competences.html** : compétences
- **experiences.html** : stages + projets
- **manifestations.html** : événements / formations
- **matieres.html** : matières + TP/projets
- **passions.html** : passions / qualités
- **quiz.html** : quiz (JS) + note + réponses correctes
- **contact.html** : formulaire de contact (mailto)
- **temoignage-iit.html** : témoignage IIT

## Lancer en local
Ouvre `index.html` dans ton navigateur.

## Héberger sur GitHub Pages (rapide)
1. Crée un repo GitHub (ex: `site-personnel-eya`).
2. Mets tous les fichiers du dossier dans le repo.
3. Settings → Pages → Deploy from branch → Branch: `main` / root.
4. Ton site sera accessible via l’URL donnée par GitHub Pages.

## Personnalisation
- Photo : remplace l’avatar par une image dans `assets/img/`.
- Pages "Passions" / "Manifestations" / "Matières" : complète le contenu si besoin.
